Dia 5: Criacao dos Fluxos

---
 	- Login:
 		Frontend(React) pega os dados .json de login - Email, Senha - manda pro backend -> Serializer + Validacao -> is_authenticate + Token de acesso JWT
React (LoginForm.tsx)
→ POST /api/auth/login/
→ Backend (APIView / Serializer)
→ authenticate(username/email, password)
→ Geração de Access Token + Refresh Token (JWT)
→ Retorno JSON para o frontend

---

 	- Logout:
 		Frontend(React) envia os tokens para o backend que valida usando o permission_classes e usa blacklist para invalidar o token
React
→ POST /api/auth/logout/
→ Envia o refresh_token no body
→ Backend:
   - permission_classes = [IsAuthenticated]
   - Blacklist do refresh_token
→ Retorna 200 OK

Criação de Item + Edição + Delete
 
Usuario clica em + para abrir um modal (URL continua a mesma)
Primeiro escolhe os dados referentes (Product)
Depois escolha as imagens (ImageProdut)
Por fim Publica o item
frostystore.com/dashboard/new-item/

Opcao de editar
Ao ir no perfil clica na foto e abre o modal porem com a URL modificada exemplo instagram.com/p/DSlmCc2gNao/, tem a foto e a descrição da foto, tem a opção de 3 pontinho no canto superior direto para Excluir ou Editar foto
Editar: quando clica em Editar abre um modal (em cima do outro, porem msm url) com as opções para editar ao clicar em salvar, o post foi editado
Excluir: abre uma telinha perguntando se tem certeza, clica em excluir e o post some
